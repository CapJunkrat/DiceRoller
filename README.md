# DiceRoller

语言 / Language: [English](README_en.md) | 中文（简体）

这是一个用于掷骰子的 Android 应用，面向需要快速掷骰与记录结果的桌游/角色扮演玩家。

项目简介：
- 支持标准骰子记法（例如 `2d6+3`、`d20`、`1d100-5`），并能解析与计算公式。
- 支持 Advantage / Disadvantage（掷两次取高/低）的掷骰模式。
- 可保存“Action Card”（预设公式），并支持可变数量/修正值的系统骰（例如 `D6`、`D20`）。
- 支持会话（Session）与历史记录，能将掷骰记录保存到本地数据库（Room）。
- 提供多种视觉风格（`DiceStyle`），并包含掷骰动画与结果分解显示。
- 集成激励型广告（AdMob Rewarded Ads）。
